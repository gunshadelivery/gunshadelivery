# Gunsha Cannabis Online Ordering

ระบบสั่งซื้อสินค้าออนไลน์สำหรับร้าน Gunsha Cannabis เชื่อมต่อกับ Line OA

## ฟีเจอร์หลัก (v1.2.1)
- ⚡ **Extreme Convenience Checkout**: สั่งซื้อรวดเร็ว ใช้แค่เบอร์โทรและปักหมุดตำแหน่ง (ไม่ต้องพิมพ์ชื่อ/ที่อยู่)
- 📍 **Smart Location Pinning**: ปุ่มดึงพิกัด GPS ปัจจุบันเพื่อความแม่นยำในการจัดส่ง
- 🚀 **Image Optimization**: บีบอัดรูปภาพอัตโนมัติ (Retina + WebP) เพื่อการโหลดที่เร็วที่สุด
- 📊 **Admin Dashboard**: ระบบสรุปยอดขาย กราฟรายวัน และจัดการสต็อกแบบ Real-time
- 🧬 **AI Strain Classifier**: แยกประเภทสายพันธุ์ (Indica/Sativa/Hybrid) อัตโนมัติจากชื่อสายพันธุ์

## การใช้งาน (สำหรับลูกค้า)
1. เลือกสินค้าที่ต้องการ
2. กดตะกร้าสินค้า
3. ระบบจะขอเพียง **เบอร์โทร** และ **ปักหมุดพิกัด**
4. แนบสลิปโอนเงิน แล้วกดสั่งซื้อ!

## การติดตั้งและใช้งาน (Local)
เพียงเปิดไฟล์ `index.html` ในเบราว์เซอร์ก็สามารถใช้งานได้ทันที
