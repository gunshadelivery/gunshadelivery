# 📱 Promptpay System - ระบบการชำระเงิน

## 🎯 ฟีเจอร์ที่เพิ่มเข้ามา

### 1. สำหรับเจ้าของร้าน (Admin Panel)
✅ **⚡ Promptpay Tab** - แท็บใหม่ในหน้า Admin สำหรับจัดการข้อมูลการชำระเงิน

#### ฟีเจอร์:
- **เพิ่มบัญชี Promptpay** 
  - ใส่ชื่อบัญชี
  - เลือกธนาคาร (รองรับ 8 ธนาคารหลัก)
  - ใส่เบอร์โทรศัพท์/หมายเลขบัญชี
  - อัพโหลด QR Code (จะอัพโหลดไปที่ ImgBB โดยอัตโนมัติ)
  - เปิด/ปิด การใช้งาน

- **แก้ไข** บัญชี Promptpay ที่มีอยู่

- **ลบ** บัญชี Promptpay (ยืนยันการลบ)

- **ตารางแสดง** ข้อมูลทั้งหมด พร้อม QR Code preview

---

### 2. สำหรับลูกค้า (Customer)
✅ **แสดงข้อมูลการชำระเงิน** ในตะกร้าก่อนแนบสลิป

#### ไฟโล์:
1. ลูกค้าเพิ่มสินค้าลงตะกร้า
2. กดปุ่ม "สั่งซื้อผ่าน LINE"
3. **ดูข้อมูลการชำระเงิน** (Promptpay accounts ที่เปิดใช้งาน)
   - ชื่อบัญชี
   - ธนาคาร
   - เบอร์โทรศัพท์
   - QR Code (สามารถกดดูขนาดใหญ่ได้)
4. ยังคงกรอก:
   - เบอร์โทรศัพท์
   - ตำแหน่ง GPS
5. **โอนเงินตามข้อมูลข้างต้น**
6. แนบสลิปโอนเงิน
7. ยืนยันและสั่งซื้อ

---

## 🔧 การตั้งค่า

### ขั้นตอน 1: ลงชื่อเข้าระบบ Admin
1. ไปที่ https://yoursite.com/admin.html
2. กรอกรหัสผ่าน (default: `gunsha888` - แก้ไขใน config.js)

### ขั้นตอน 2: เพิ่ม Promptpay Account
1. คลิกที่ tab "⚡ Promptpay"
2. กดปุ่ม "เพิ่ม Promptpay ใหม่"
3. กรอกข้อมูล:
   - **ชื่อบัญชี**: เช่น "สมชาย จันทร์เสมา"
   - **ธนาคาร**: เลือกจากรายการ
   - **เบอร์บัญชี**: 0812345678 หรือ 123-4-5678-9
   - **QR Code**: เลือกรูปจากเครื่อง (จะอัพโหลดโดยอัตโนมัติ)
   - **สถานะ**: เลือก "ใช้งาน" เพื่อให้ลูกค้าเห็น
4. กดปุ่ม "บันทึก"

### ขั้นตอน 3: ยืนยันลูกค้าเห็น
1. ไปที่หน้า https://yoursite.com/index.html
2. เพิ่มสินค้าลงตะกร้า
3. กดปุ่ม "สั่งซื้อผ่าน LINE"
4. ควรจะเห็นข้อมูล Promptpay ที่เพิ่มไป

## 📊 ข้อมูลที่เก็บ

ข้อมูล Promptpay เก็บใน **localStorage** ของบราว์เซอร์:
- **Key**: `promptpayData`
- **Format**: JSON Array

### ตัวอย่างข้อมูล:
```json
[
  {
    "name": "สมชาย จันทร์เสมา",
    "bank": "ธนาคารกรุงเทพ (BBL)",
    "number": "0812345678",
    "status": "active",
    "qrImage": "https://i.ibb.co/..."
  },
  {
    "name": "ดำรงศักดิ์ สุขสันต์",
    "bank": "ธนาคารกสิกรไทย (KBANK)",
    "number": "123-4-5678-9",
    "status": "active",
    "qrImage": "https://i.ibb.co/..."
  }
]
```

## ⚙️ การแก้ไขและลบ

### แก้ไข Promptpay
1. ไปที่ Admin → Promptpay tab
2. กดไอคอน ✏️ (แก้ไข)
3. แก้ไขข้อมูลตามต้องการ
4. กดปุ่ม "บันทึก"

### ลบ Promptpay
1. ไปที่ Admin → Promptpay tab
2. กดไอคอน 🗑️ (ลบ)
3. ยืนยันการลบ
4. ข้อมูลจะถูกลบออกจาก localStorage

### ปิด/เปิดใช้งาน
- ในการแก้ไข ให้เปลี่ยน "สถานะ" เป็น "ไม่ใช้งาน" ลูกค้าจะไม่เห็นบัญชีนี้
- เปลี่ยนกลับเป็น "ใช้งาน" เพื่อให้ลูกค้าเห็นอีกครั้ง

## 🐛 การแก้ไขปัญหาธรรมดา

### ปัญหา: ข้อมูล Promptpay หายไป
**วิธีแก้**: Browser localStorage อาจถูกล้างทำความสะอาด
- ลองใช้ Private/Incognito mode
- หรือลบข้อมูลบราว์เซอร์เฉพาะ localStorage
- เพิ่มข้อมูล Promptpay ใหม่อีกครั้ง

### ปัญหา: QR Code ไม่แสดง
**วิธีแก้**: 
- ตรวจสอบว่า ImgBB API key ถูกต้อง (ใน config.js)
- ลองอัพโหลด QR Code ใหม่อีกครั้ง
- ตรวจสอบสถานะ "ใช้งาน"

### ปัญหา: ลูกค้าไม่เห็นข้อมูลการชำระเงิน
**วิธีแก้**: 
- ตรวจสอบว่า Promptpay account มี status = "active"
- ล้าง browser cache (Ctrl+Shift+Del)
- Reload หน้าใหม่
